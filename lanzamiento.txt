Comandos para arrancar el proyecto:

- npm install -g (Trae todas las dependencias)
- npm install -g browserify
- npm install gulp-cli -g

- browserify assets/js/vendor.js --o dist/vendor.js (Compila los js de vendors) 

- gulp vendorCss (Compila los css de vendors)

- gulp watch (Te mira los .scss y los compila en style.css)
